# VX 플랫폼 개발 Task 리스트

## 프로젝트 개요
- **플랫폼명**: VX_리뷰X (참여 기반 캠페인 플랫폼)
- **기술 스택**: Spring Boot + Thymeleaf + MySQL
- **개발 우선순위**: 구매평 캠페인 프로토타입 먼저 구현
- **팀 구성**: 1인 개발 (바이브코더 + Claude)

## Phase 1: 프로토타입 개발 (구매평 캠페인 중심)

### P001. 프로젝트 초기 설정 (Spring Boot + Thymeleaf + MySQL 기반)
- **P001-1**: Spring Boot 프로젝트 생성 및 기본 설정
  - Spring Boot 3.x 버전 사용
  - Maven/Gradle 빌드 도구 선택
  - 기본 의존성 추가 (Web, JPA, MySQL, Thymeleaf, Security)
- **P001-2**: MySQL 연결 및 JPA 설정
  - application.properties/yml 설정
  - 데이터베이스 연결 테스트
- **P001-3**: Thymeleaf 템플릿 엔진 설정
  - 기본 템플릿 구조 설정
  - 정적 리소스 설정
- **P001-4**: 프로젝트 구조 및 패키지 설계
  - MVC 패턴 기반 패키지 구조
  - 도메인별 패키지 분리

### P002. 데이터베이스 설계 (구매평 캠페인 중심)
- **P002-1**: 사용자 테이블 설계 (참여자, 파트너, 관리자)
  - User 테이블: 공통 사용자 정보
  - Role 기반 권한 관리
- **P002-2**: 캠페인 테이블 설계 (구매평 캠페인 우선)
  - Campaign 테이블: 캠페인 기본 정보
  - 구매평 캠페인 특화 필드
- **P002-3**: 리뷰 테이블 설계 (텍스트/포토 리뷰)
  - Review 테이블: 리뷰 기본 정보
  - ReviewFile 테이블: 첨부 파일 관리
- **P002-4**: 정산 관련 테이블 설계
  - Point 테이블: 포인트 적립/사용 내역
  - Withdrawal 테이블: 출금 신청 관리
- **P002-5**: Entity 클래스 생성 및 관계 매핑
  - JPA Entity 클래스 생성
  - 연관관계 매핑 설정

### P003. 기본 인증 시스템 구현 (카카오 로그인은 나중에)
- **P003-1**: Spring Security 설정
  - 기본 보안 설정
  - 패스워드 암호화
- **P003-2**: 사용자 회원가입/로그인 기능
  - 회원가입 폼 및 검증
  - 로그인 처리
- **P003-3**: 역할 기반 권한 관리 (참여자/파트너/관리자)
  - 권한별 접근 제어
  - 메뉴 및 기능 제한

### P004. 구매평 캠페인 핵심 기능 구현
- **P004-1**: 파트너 - 구매평 캠페인 등록 기능
  - 캠페인 등록 폼
  - 캠페인 정보 저장
- **P004-2**: 참여자 - 캠페인 목록 조회 및 참여 신청
  - 캠페인 목록 페이지
  - 참여 신청 처리
- **P004-3**: 참여자 - 구매평 리뷰 등록 (텍스트/포토)
  - 리뷰 등록 폼
  - 구매 영수증 업로드
  - 텍스트/포토 리뷰 작성
- **P004-4**: 파트너 - 리뷰 승인/반려 처리
  - 리뷰 목록 조회
  - 승인/반려 처리 기능

### P005. 파일 업로드 시스템 (이미지 처리)
- **P005-1**: 로컬 파일 업로드 기능 구현
  - MultipartFile 처리
  - 파일 저장 경로 설정
- **P005-2**: 이미지 검증 로직 (5MB 제한, jpg/png)
  - 파일 크기 제한
  - 확장자 검증
- **P005-3**: 이미지 썸네일 생성 (선택사항)
  - 이미지 리사이징
  - 썸네일 생성

### P006. 기본 정산 시스템
- **P006-1**: 포인트 적립 시스템
  - 리뷰 승인 시 포인트 적립
  - 포인트 내역 관리
- **P006-2**: 출금 신청 기능 (간단한 형태)
  - 출금 신청 폼
  - 계좌 정보 등록
- **P006-3**: 관리자 정산 승인 기능
  - 출금 신청 목록
  - 승인/반려 처리

### P007. 관리 기능 구현
- **P007-1**: 관리자 대시보드 (기본 현황)
  - 전체 현황 요약
  - 실시간 통계
- **P007-2**: 캠페인 관리 (승인/반려/상태변경)
  - 캠페인 목록 관리
  - 상태 변경 기능
- **P007-3**: 사용자 관리 (목록, 상태변경)
  - 사용자 목록 조회
  - 사용자 상태 관리

### P008. UI 개발 (Thymeleaf + Bootstrap)
- **P008-1**: 공통 레이아웃 및 네비게이션
  - 기본 레이아웃 템플릿
  - 반응형 네비게이션
- **P008-2**: 참여자 페이지 (캠페인 목록, 마이페이지)
  - 캠페인 목록 UI
  - 마이페이지 UI
- **P008-3**: 파트너 페이지 (캠페인 관리, 리뷰 관리)
  - 캠페인 관리 UI
  - 리뷰 관리 UI
- **P008-4**: 관리자 페이지 (대시보드, 관리 기능)
  - 대시보드 UI
  - 관리 기능 UI

### P009. 테스트 및 버그 수정
